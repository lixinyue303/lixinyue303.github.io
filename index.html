Markdown语法

1.标题：
几个#就是几级标题
2.段落：
两个空格以上加回车，表示换行
或者段落后面使用一个空行表示重新开始一个段落
3.列表：
有序列表：数字+. 比如1. 2. 3.
无序列表：* or + or -
列表嵌套：
在字列表的选项前加四个空格
4.区块：
使用符号>
>最外层
>>第一层嵌套
>>>第二层嵌套
5.代码
代码区块： 反引号制表符 6.图片 语法格式： 7.表格 Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。 语法格式如下：| 表头 | 表头 | | —- | —- | | 单元格 | 单元格 | | 单元格 | 单元格 |`
